# CEiVoice

Support AI-enhanced Support Ticket Management System

---

## Setup

### 1. Start Docker

From project root: 

```bash
docker compose up --build
```
---

### 2. Start Frontend
```bash
cd frontend
npm install
npm run dev
```
---

### 3. Start Backend
```bash
cd backend
npm install
node server.js
```
---

### Access URLs
Frontend:
http://localhost:5173

phpMyAdmin:
http://localhost:8081
